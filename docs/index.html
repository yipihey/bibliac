Everything You Need for Research

Built by researchers, for researchers

NASA ADS Integration

Search and sync with the Astrophysics Data System. Import papers by bibcode, DOI, or arXiv ID. Get complete metadata automatically.

Smart PDF Management

Download PDFs from multiple sources: arXiv, publishers, ADS scans. Keep different versions organized. Add highlights and annotations.

References & Citations

View what papers cite and are cited by your papers. One-click import of references to build your reading list.

Collections & Organization

Create nested collections to organize your library. Use smart searches to automatically filter papers by criteria.

BibTeX Export

Export your entire library or selected papers to BibTeX. Copy citation keys with keyboard shortcuts. Auto-generate master.bib files.

iCloud Sync

Your library syncs across all your Macs via iCloud. Multiple libraries supported. Never lose your paper collection.

Keyboard First

Navigate your library with vim-style shortcuts. Quick actions for common tasks. Designed for power users.

Open Source

MIT licensed and built in the open. Contribute features, report bugs, or fork it for your needs.

Get Started in Minutes

Simple setup for macOS

1

Download

Click the download button above to get the latest DMG file from GitHub Releases.

2

Install

Open the DMG and drag Bibliac to your Applications folder. Important: Right-click the app and select "Open" the first time (macOS beta app security).

3

Get ADS Token

Get your API token from NASA ADS Settings. Paste it in Preferences.

4

Start Reading

Import papers by dragging PDFs, pasting bibcodes, or searching ADS directly in the app.

System Requirements

  • macOS 11.0 (Big Sur) or later
  • Apple Silicon (M1/M2/M3) or Intel Mac
  • ~100 MB disk space
  • NASA ADS account (free) for full functionality
